# dwm-profile

This is my dwm profile. Still a work in progress but getting there

## Dot Files:
<ul>
<li> .xinitrc belongs to ~/
<li> settings.ini belongs to ~/.config/gtk-3.0/
<li> slock.service belongs to /etc/systemd/system/
</ul>
